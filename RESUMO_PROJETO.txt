3. PÁGINAS DA APLICAÇÃO
   ✅ index.html - Página principal com 3 abas:
      - Agendar Visita (formulário)
      - Calendário (visualização de horários)
      - Agendamentos (lista de agendamentos)
   
   ✅ admin.html - Painel administrativo com:
      - Tela de login
      - Gerenciamento de agendamentos
      - Gerenciamento de horários

4. BANCO DE DADOS
   ✅ Tabela: agendamentos
      - Armazena dados das escolas e visitas agendadas
      - Campos: escola, email, telefone, data, horário, turmas, observações
   
   ✅ Tabela: horarios_disponiveis
      - Armazena horários disponíveis
      - Marca automaticamente como ocupado quando agendado

5. API ENDPOINTS
   ✅ GET  /api/horarios - Listar horários disponíveis
   ✅ GET  /api/agendamentos - Listar agendamentos
   ✅ POST /api/agendamentos - Criar novo agendamento
   ✅ POST /api/horarios - Adicionar horário (admin)
   ✅ DELETE /api/agendamentos/:id - Deletar agendamento (admin)
   ✅ POST /api/login - Autenticação admin

6. SEGURANÇA
   ✅ Autenticação JWT para rotas protegidas
   ✅ Validação de dados no backend
   ✅ CORS habilitado para requisições
   ✅ Senhas de admin criptografadas
